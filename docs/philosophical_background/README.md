# 哲学背景：项目的「内核」

> **这个目录，是整个项目真正的源头。**

很多人会以为「人类知识地基计划」是主角，而这些哲学思考只是顺带的背景。**这是反的。**

- **内核（源头）** = 对「智能的本质」的追问。
- **产物（落地）** = 「人类知识地基计划」，它是上述追问推到尽头后，逼出来的一个具体工程。

如果你想知道这个项目**「为什么是这件事，而不是别的事」**、它的野心到底有多大，请按下面的顺序读完这四篇。读完你就会明白：仓库里每一个工程动作（探针、假设卡片、失效边界），背后都站着一整套世界观。

## 推荐阅读顺序

1. **[00 项目缘起：从「智能的本质」到「人类知识地基」](00_项目缘起_从智能的本质到知识地基.md)**
   先建立"内核 → 产物"的全局地图。把"为什么要做这个项目"的来龙去脉一次讲透。**强烈建议从这篇开始。**

2. **[01 智能的本质：最终精要](01_智能的本质_最终精要.md)**
   内核的完整展开——十余万字极限思辨的结晶。智能即效率、地基—建筑模型、冲突驱动重构、意识祛魅、双流架构。

3. **[02 理解的新定义：解耦与因果](02_理解的新定义_解耦与因果.md)**
   最新、最硬核的一块拼图：**理解 = 解耦**。它直接回答了"挖隐含假设为什么就是在加深理解"，是连接内核与产物的关键一环。

4. **[03 体验是价值权重：痛苦快乐不是玄学](03_体验是价值权重_痛苦快乐不是玄学.md)**
   内核的一块延伸：连最像"玄学"的主观体验，也能被还原成可分析的信息结构。它示范了本项目一贯的方法论。

## 阅读心态

> **请始终带着"淘金者心态"阅读。**
>
> 这些思考源自直觉驱动的个人探索，是"科学正规军到来前的前哨小分队"，难免有不严密、甚至"民科"式的大胆猜测。
> 不要用阅读教科书的标准来求全责备，而要像在矿堆里翻找发光物品一样，敏锐地捕捉那些有价值的认知火花，再用科学工程的方法去提纯验证。
