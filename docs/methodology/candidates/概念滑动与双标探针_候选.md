# 候选探针（待证）：【概念滑动 / 双标探针】

> 状态：**候选 / 待证**，暂不给正式编号。
> 原因：它的“假阳性风险”很高（见下），必须先经受考验，才能转正进入正式探针体系。

## 它想捕捉什么

一种藏在“表达层”的隐性假设：

> **“这个词，从头到尾指的是同一个东西。”**

而真相往往是：一个关键词**悄悄扛了两个意思**，整段论证的“说服力”，恰恰来自在这两个意思之间**偷偷换脚**。

## 两种等价讲法

* **学术讲法**：概念滑动 / 偷换概念（equivocation）——同一个词，两个意思。
* **大白话讲法（推荐）**：**双标**——同一场比较里，前后用了两套不同的衡量标准。

## 命中案例

* **“机制即天花板”**：“机制”一词在“被取用的那部分能力”和“结构的全部组合能力”之间滑动。（详见 `06_基底决定论探针.md`）
* **“整体大于部分之和”**：“之和”一词在“零件独立相加”和“零件+关系一起算”之间玩双标。（详见 `07_孪生陷阱_整体与部分.md`）

## 转正前必须满足的铁律（防假阳性）

本候选探针**假阳性风险极高**——因为“你这个词有歧义”这种指控，几乎能扣在任何论述上。因此，每一次使用，**必须**满足：

1. **摆出两个具体定义**：明确写出这个词实际承载的意思 A 和意思 B，不许只说“它模糊”。
2. **指认换脚动作**：具体指出某个论证是如何**只有靠在 A、B 之间换脚才能成立**的。
3. **证明危害**：说明这次滑动导致了什么错误结论或无效争论。

做不到以上三点，**不算命中**，不得据此宣称发现了隐性前提。

## 已登记的合格命中（转正进度）

1. **“整体大于部分之和” 借力 “写不尽”**（见 `07_孪生陷阱_整体与部分.md` 第六节）
   * 两个定义：意思 A =“逻辑上超出（>）”；意思 B =“现实中关系写不尽（资源约束）”。
   * 换脚动作：用 B（现实写不尽）去支撑 A（逻辑大于）。
   * 危害：把一个资源约束问题，误判成了逻辑/本体层面的“涌现”，从而错误地否定了“整体 = 部分 + 关系”这个等式。
   * 进度：**1 / 3**。

## 转正条件

当本候选探针积累到**至少 3 个**满足上述铁律、且证据等级达到 B 级以上的独立案例时，方可申请转正、进入正式探针编号体系。
