# 人类知识地基计划：探矿工程学方法论 (Methodology)

## 核心理念：从“炼金术”走向“现代地质勘探”

过去，人类寻找知识体系底层裂缝的方式，高度依赖于“百年一遇的天才”的灵光一现（如爱因斯坦、黎曼、达尔文）。这种方式本质上类似于前科学时代的“炼金术”或“碰运气”。

**“人类知识地基计划”的核心目标，是将这种“范式转移”的过程工程化、系统化。**

我们认为，基于《智能的本质》中“智能是资源受限条件下的效率优化机制”这一核心理论，历史上所有被打破的“隐性前提”，本质上都是当年为了“省资源、图省事”而欠下的技术债。

因此，寻找隐性前提不需要靠运气，而是可以通过建立一套**“探矿工程学”**来进行系统扫描。

## 方法论闭环

我们建立了一个正反馈的闭环系统：

1. **历史案例拆解**：系统性地拆解科学史与技术史上已经成功的“挖地基”案例。
2. **提取共同模式**：从历史案例中，提炼出隐性前提崩塌的常见模式（即“探针”）。
3. **形成扫描方法**：将这些模式转化为方法论，并充分利用 AI 可执行的结构化 Prompt 和扫描策略。
4. **扫描新领域**：利用 AI 探针，对当前的学科（如 AI、软件工程、医学、经济学）进行地毯式扫描。
5. **证据闸门验证**：用五道证据闸门和 A/B/C 置信等级，防止把普通裂纹夸成地震。
6. **证伪自检**：用证伪协议给自己照镜子，强制收集反例、做活体预测、请专家打脸，确保探针有“鉴别力”而非“见谁都喊有病”。
7. **发现与验证**：找到候选的隐性前提，并用历史模式进行交叉验证，最终输出“地基研究报告”。

## 目录结构

* `00_物理真相锁定探针.md`：元探针（Meta-Probe）。所有隐性前提的底层根源——把“当前观测/实现手段的物理极限”直接等同于“事物本身的物理真相”。
* `01_三大探针系统.md`：目前提炼出的核心扫描工具（妥协遗忘、地图误认、补丁坍塌）。
* `02_历史案例拆解矩阵.md`：用于拆解历史范式转移的标准模板与案例库。
* `03_证据闸门与置信等级.md`：案例入库前的反幻觉验证协议，包含五道证据闸门、A/B/C 置信等级和隐性前提卡片模板。
* `04_证伪协议.md`：项目的“照妖镜”。用大白话讲清楚本方法最可能犯的四个错误（幸存者偏差、假阳性过高、用造方法论逃避考试、理论变万能咒语），并设定三个对外发布前必须完成的硬性任务。
* `05_探针输出升级_条件与边界.md`：一次重要的方法论迭代。探针输出从“要挖/不用挖”的开关，升级为“①成立条件 ②条件是否还在 ③在哪些新场景已消失”的边界地图。
* `06_基底决定论探针.md`：第五个探针（纵向层级混淆）。捕捉“用底层机制的描述去断言高层能力上限”的错误——核心是把“被取用的那部分能力”误当成了“原始结构的全部组合能力”。
* `07_孪生陷阱_整体与部分.md`：基底决定论的“孪生补充”。用“双标”讲清“整体大于部分之和”的真相（整体 = 部分 + 组织关系），并破掉还原论与玄学整体论两个对称陷阱。
* `cases/`：已完成结构化拆解的经典案例库（QWERTY、24 帧、代码即文本、二进制vs三进制、Transformer虚拟机等）。
* `candidates/`：候选探针区（待证，未转正）。当前有【概念滑动 / 双标探针】，因假阳性风险高，需积累足够合格案例方可转正。
