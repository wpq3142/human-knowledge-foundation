# 核心工具：三大 AI 探针系统 (The 3 Probes)

> **注意**：本目录下的 `00_物理真相锁定探针.md` 是本方法论的**元探针（Meta-Probe）**，它试图回答所有隐性前提的底层根源问题。以下三大探针是其在工程实践中的具体表现形式。

基于“智能是资源受限条件下的效率优化机制”这一核心理论，我们提炼出了三个用于扫描隐性前提的“探针”。未来，AI Agent 将携带这些探针，对各个学科的论文、代码库、行业标准进行系统性扫描。

## 探针 1：【妥协遗忘探针】 (The "Forgotten Compromise" Probe)

* **检测原理**：寻找那些“当年因为硬件/算力/观测条件/通信带宽受限而做出的降维妥协，但在限制解除后，却被当成了不可动摇的真理”的现象。
* **理论依据**：效率妥协一旦固化为基础设施，后人往往会遗忘其“妥协”的本质，将其视为“客观规律”。
* **历史命中案例**：
  * QWERTY 键盘布局（本是为了防止机械打字机卡键而故意降低打字速度，后来却成了全人类的输入标准）。
  * 电影的 24 帧（本是为了省胶片，后来被奉为“电影感”和艺术标准）。
  * 软件工程的“代码即纯文本”（本是为了适应 70 年代的终端输出，后来变成了 Git 和代码管理的基础）。
* **AI 扫描指令示例**：
  > `"在这个领域中，有哪些核心标准、数据结构或基础工作流，其最初的设计是为了适应 20 年前或更早的硬件/网络/观测限制？这些限制今天是否依然存在？"`

## 探针 2：【地图误认探针】 (The "Map vs. Territory" Probe)

* **检测原理**：寻找那些“把测量指标、表达工具、UI 投影，错误地当成了事物本体”的领域。
* **理论依据**：人类为了降低认知成本，倾向于用简单的指标或投影来替代复杂的现实。久而久之，“地图”替代了“领土”。
* **历史命中案例**：
  * 把“智商测试分数”当成智力本体。
  * 把“GDP”当成社会发展本体。
  * 把“自然语言文本”当成知识本体。
* **AI 扫描指令示例**：
  > `"该领域目前最重要的 KPI 或核心测量指标是什么？在哪些极端边缘案例中，该指标极高，但实际系统已经崩溃或毫无价值（即发生指标反噬）？"`

## 探针 3：【补丁坍塌探针】 (The "Patchwork Collapse" Probe)

* **检测原理**：寻找那些“为了解释新现象或解决新问题，不惜引入极其复杂、丑陋的局部补丁，导致系统臃肿不堪”的理论或工程体系。
* **理论依据**：当地基出现裂缝时，系统最初的反应一定是做“局部修复”。当局部修复的成本超过了重构地基的成本时，范式转移就会发生。
* **历史命中案例**：
  * 地心说后期的“本轮与均轮”（为了强行解释行星逆行，画了无数个圈，极其复杂）。
  * 当前大语言模型为了解决幻觉问题，而堆砌的无数 Prompt 技巧、复杂的 RAG 检索流程和外挂工具链。
* **AI 扫描指令示例**：
  > `"该领域在过去 5 年内，为了解决某一个顽固缺陷，是否衍生出了极其庞大且复杂的周边工具链或修补理论？这些修补是否让整个系统变得越来越难以理解？"`
